# Bloomburrow Project Roadmap

This file tracks major development areas for the Bloomburrow campaign setting.

## Areas for Improvement

Here is a list of potential areas to flesh out, based on an analysis of the existing material.

### 1. Detailed Faction Guides
- **Goal:** Expand on the existing factions (e.g., Stormchasers, Plumecreed, Brightblades) to make them fully interactive parts of the campaign.
- **Tasks:**
    - Define clear goals, motivations, allies, and enemies for each faction.
    - Create a system for player membership, including ranks, renown, and mechanical benefits.
    - Develop faction-specific quest hooks and NPCs.

### 2. In-Depth Location Guides
- **Goal:** Transform key locations from simple descriptions into fully detailed, runnable adventure sites.
- **Tasks:**
    - Select a major settlement (e.g., Three Tree City, Fountainport) to serve as a pilot.
    - Create a detailed map for the chosen location.
    - Populate the location with key NPCs, points of interest (shops, taverns, etc.), and local plot hooks.
    - Develop location-specific random encounter tables.

### 3. Unique Magic Items
- **Goal:** Create a collection of unique magic items that are deeply rooted in the lore and cultures of Bloomburrow.
- **Tasks:**
    - Design 5-10 new magic items.
    - For each item, write its D\&D 5e statistics (rarity, attunement, abilities).
    - Write a short lore description for each item, tying it to a specific species, faction, or historical event.

### 4. Continue JSON-to-Markdown Conversion
- **Goal:** Make all game elements accessible and easy to read in a consistent Markdown format.
- **Tasks:**
    - Convert the custom **Spells** from `Bloomburrow.json` into a dedicated Markdown file.
    - Convert the custom **Feats** from `Bloomburrow.json` into a dedicated Markdown file.
    - Convert the custom **Backgrounds** from `Bloomburrow.json` into individual Markdown files.
